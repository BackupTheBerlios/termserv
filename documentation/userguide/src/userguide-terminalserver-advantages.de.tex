\section{Die Vorteile}
 
Der Terminal-Server reizt die Vorteile eines Netzwerks voll aus. Konsequent eingesetzt, verringert er
den Verwaltungsaufwand des Systemamdministrators enorm. Aber nicht nur das:
 
\begin{itemize}
\item Ältere Hardware kann für die Terminals weiter verwendet werden.
\item Es braucht nur noch ein Rechner -- der Server -- administriert zu werden.
\item Software wird nur einmal zentral aufgespielt, gewartet und steht allen angeschlossenen Terminals zur Verfügung.
\item Zeit und Kosten werden erheblich minimiert.
\end{itemize}
